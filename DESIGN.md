## 一、项目概述

**类型**：修仙题材卡牌肉鸽（Roguelike Deckbuilder）
**平台**：网页单文件 HTML（未来可移植微信小程序）
**技术栈**：Vanilla JS + CSS（零依赖）
**核心循环**：外围养成（洞府）→ 卡牌肉鸽副本（幽冥谷）→ 材料回归 → 养成强化

### 游戏流程

```
[首次进入] → 灵根觉醒（随机五行）
     ↓
[洞府/主界面] ←—————————————┐
  ├── ⚔️ 历练 → 准备丹药 → 幽冥谷副本 → 通关/死亡 → 返回洞府
  ├── 🧘 修炼 → 积攒修为 → 突破境界（三选一神通）
  ├── ⚗️ 炼丹 → 消耗材料 → 炼制丹药（装备生效）
  ├── 🔨 炼器 → 消耗材料 → 炼制法器（一阶→二阶升级）
  ├── 📖 功法 → 学习/装备功法（改变牌组+被动加成）
  ├── ⚡ 神通 → 装备被动战斗技能（最多3个）
  ├── 🌅 悟道 → 消耗200修为领悟神通（三选一）
  └── 📦 背包 → 查看装备/材料/重置存档
```

## 二、角色系统

### 2.1 灵根（五行）

首次进入游戏随机分配，决定初始优势和专属卡牌。

| 灵根 | 颜色 | 初始加成 | 专属卡牌 |
|------|------|----------|----------|
| 🔥 火 | #e74c3c | 攻击牌+1伤害，多1张火鸦 | — |
| 🌿 木 | #2ecc71 | 治疗效果+2，多1张灵草护体 | — |
| ⛰️ 土 | #f39c12 | 开局+3格挡，格挡牌+2 | — |
| 💧 水 | #3498db | 初始灵力4，首回合多抽1张 | — |
| ⚔️ 金 | #bdc3c7 | 10%暴击（双倍伤害），多1张金刃符 | — |

### 2.2 境界体系

共15个境界等级，每层需积攒修为突破。

```
炼气1层(0xp) → 2层(100) → 3层(200) → 4层(350) → 5层(500)
→ 6层(700) → 7层(1000) → 8层(1400) → 9层(1900)
→ 筑基初期(2500) → 中期(3200) → 后期(4000)
→ 金丹初期(5000) → 中期(6200) → 后期(7500)
```

**突破消耗：**
- 破层（每层）：💊 突破丹 ×1
- 破境（每3层）：✨ 筑基丹 ×1

**突破奖励：**
- 每次突破：maxHp +5
- 每3层突破：maxSp +1
- 突破时三选一获得1个神通

### 2.3 持久化存档

使用 localStorage（key: `xianxia_save`），保存内容：
- 灵根、境界、修为
- 材料（灵草/矿石/妖丹）
- 突破丹/筑基丹库存
- 已学习功法、已装备功法
- 已领悟神通、已装备神通
- 已拥有丹药/法器及装备状态
- 历练卡牌收藏
- 悟道冷却时间

```typescript

  id: number;          // 唯一ID
```

## 七、战斗系统

### 7.1 战斗流程

```
startTurn()
  → 灵力恢复、格挡清零、神通回合初始化
  → 抽牌（5张 + 水灵根/聚灵佩加成）
  → 显示敌人意图

[玩家出牌阶段]
  → _playCard() → 费用扣除 → 神通触发检查 → 效果结算
  → 可用丹药（每场1次）
  → 点击「结束回合」

endPlayerTurn()
  → 弃手牌
  → 灼烧tick（伤害+层数-1，百毒侵不掉）
  → 敌人行动（意图结算）
  → 以柔克刚检查
  → 不灭金身检查
  → 中毒tick
  → 涅槃检查（死亡复活）
  → 蓄势追踪
```

### 7.2 伤害计算

```
总伤害 = 基础伤害
  + 火鸦加成（火鸦变）
  + nextAtk（淬火术/凝元丹）
  + 丹意层数
  + 灵根加成（火+1）
  + 功法加成
  + 法器加成（玄铁剑）
  + 神通加成（舍身/烈焰之心/弱点洞悉）

乘法修正：
  × 焚天（第3张火翻倍）
  × 金锐（15%翻倍）
  × 灵犀戒暴击（5%）
  × 金灵根暴击（10%）
  × 连环诀（≥4张攻击+50%）
```

### 7.3 状态效果

| 状态 | 触发时机 | 效果 |
|------|----------|------|
| 灼烧(burn) | 回合结束（出牌后） | 减少HP，层数-1 |
| 中毒(poison) | 回合结束（敌人行动后） | 减少HP，层数-1（百毒侵不掉） |
| 格挡(block) | 回合开始清零 | 抵挡伤害 |
| 攻击加成(nextAtk) | 攻击时消耗 | 叠加伤害 |

### 7.4 丹药（战斗消耗品）

进入副本前选择携带（总量上限5瓶）。

| 丹药 | 效果 | 上限 |
|------|------|------|
| 回春丹 | 回复15HP | 3 |
| 凝元丹 | 下次攻击+3 | 2 |
| 护脉丹 | 获得10格挡 | 2 |

### 7.5 连击系统

同一玩家回合内同一张牌打出≥2次时触发连击弹出动画。

## 十二、设计原则

1. **外围养成驱动重玩** — 每次历练获得材料，回来强化角色，下次走得更远
2. **多系统联动** — 境界→神通、材料→丹药/法器、功法→牌组、灵根→全链路
3. **选择即策略** — 丹药装备上限、法器升级消耗、功法互斥、神通位有限
4. **渐进难度** — 敌人随层数缩放、Boss狂暴线、后期需综合养成才能通关
