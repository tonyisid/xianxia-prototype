## 一、项目概述

**类型**：修仙题材卡牌肉鸽（Roguelike Deckbuilder）
**平台**：网页单文件 HTML（未来可移植微信小程序）
**技术栈**：Vanilla JS + CSS（零依赖）
**核心循环**：外围养成（洞府）→ 卡牌肉鸽副本（幽冥谷）→ 材料回归 → 养成强化

### 游戏流程

```
[首次进入] → 灵根觉醒（随机五行）
     ↓
[洞府/主界面] ←—————————————┐
  ├── ⚔️ 历练 → 准备丹药 → 幽冥谷副本 → 通关/死亡 → 返回洞府
  ├── 🧘 修炼 → 积攒修为 → 突破境界（三选一神通）
  ├── ⚗️ 炼丹 → 消耗材料 → 炼制丹药（装备生效）
  ├── 🔨 炼器 → 消耗材料 → 炼制法器（一阶→二阶升级）
  ├── 📖 功法阁 → 查看50+功法卡池/已收集卡牌
  ├── ⚡ 神通 → 装备被动战斗技能（最多3个）
  ├── 🌅 悟道 → 消耗200修为领悟神通（三选一）
  └── 📦 背包 → 查看装备/材料/重置存档
```

## 二、角色系统

### 2.1 灵根（五行）

首次进入游戏随机分配，决定初始优势和专属卡牌。

| ⚔️ 金 | #bdc3c7 | 10%暴击（双倍伤害），多1张金刃符 | — |

### 2.2 境界体系

共15个境界等级，每层需积攒修为突破。

```
炼气1层(0xp) → 2层(100) → 3层(200) → 4层(350) → 5层(500)
→ 6层(700) → 7层(1000) → 8层(1400) → 9层(1900)
→ 筑基初期(2500) → 中期(3200) → 后期(4000)
→ 金丹初期(5000) → 中期(6200) → 后期(7500)
```

**突破消耗：**
- 破层（每层）：💊 突破丹 ×1
- 破境（每3层）：✨ 筑基丹 ×1

**突破奖励：**
- 每次突破：maxHp +5
- 每3层突破：maxSp +1
- 突破时三选一获得1个神通

### 2.3 持久化存档

使用 localStorage（key: `xianxia_save`），保存内容：
- 灵根、境界、修为
- 材料（灵草/矿石/妖丹）
- 突破丹/筑基丹库存
- 已学习功法、已装备功法
- 已领悟神通、已装备神通
- 已拥有丹药/法器及装备状态
- 历练卡牌收藏
- 悟道冷却时间

## 六、功法卡牌系统

**功法即卡牌。** 所有战斗中打出的牌都是功法，每张功法带五行属性。

### 6.1 卡牌数据结构

```typescript
{
  id: number;          // 唯一ID
  name: string;        // 名称
  type: 'atk'|'def'|'skill';  // 攻击/防御/功法
  cost: number;        // 灵力消耗
  dmg: number;         // 伤害
  blk: number;         // 格挡
  burn: number;        // 灼烧
  poison: number;      // 中毒
  draw: number;        // 抽牌
  heal: number;        // 治疗
  el: Element;         // 元素属性(fire/wood/earth/water/metal/none)
  desc: string;        // 描述
  upgraded: boolean;   // 是否升级
}
```

### 6.2 元素属性

| 元素 | 标签 | 定位 | 关联机制 |
|------|------|------|----------|
| 🔥 火(fire) | 符箓·火 | 进攻型 | 灼烧、高伤害 |
| 🌿 木(wood) | 符箓·木 | 持续型 | 中毒、回复 |
| ⛰️ 土(earth) | 符箓·土 | 防御型 | 格挡、攻防兼备 |
| 💧 水(water) | 符箓·水 | 控制型 | 抽牌、攻守平衡 |
| ⚔️ 金(metal) | 符箓·金 | 暴击型 | 高直伤、穿透 |
| 无(none) | 功法·无 | 中立 | 通用辅助 |

## 七、战斗系统

### 7.1 战斗流程

```
startTurn()
  → 灵力恢复、格挡清零、神通回合初始化
  → 抽牌（5张 + 水灵根/聚灵佩加成）
  → 显示敌人意图

[玩家出牌阶段]
  → _playCard() → 费用扣除 → 神通触发检查 → 效果结算
  → 可用丹药（每场1次）
  → 点击「结束回合」

endPlayerTurn()
  → 弃手牌
  → 灼烧tick（伤害+层数-1，百毒侵不掉）
  → 敌人行动（意图结算）
  → 以柔克刚检查
  → 不灭金身检查
  → 中毒tick
  → 涅槃检查（死亡复活）
  → 蓄势追踪
```

### 7.2 伤害计算

```
总伤害 = 基础伤害
  + 火鸦加成（火鸦变）
  + nextAtk（淬火术/凝元丹）
  + 丹意层数
  + 灵根加成（火+1）
  + 功法加成
  + 法器加成（玄铁剑）
```

## 十二、设计原则

1. **外围养成驱动重玩** — 每次历练获得材料，回来强化角色，下次走得更远
2. **多系统联动** — 境界→神通、材料→丹药/法器、功法→牌组、灵根→全链路
3. **选择即策略** — 丹药装备上限、法器升级消耗、功法互斥、神通位有限
4. **渐进难度** — 敌人随层数缩放、Boss狂暴线、后期需综合养成才能通关

### Phase 2：内容扩充（2-3月）

**目标：** 足够的内容支撑 10+ 小时游戏时长

#### 2.1 功法扩充（150+张）

当前50张 → 目标150+张，每个元素25-30张：

| 稀有度 | 数量 | 获取方式 |
|--------|------|----------|
| 白(普通) | 60 | 普通战斗、商店 |
| 绿(进阶) | 50 | 精英战斗、事件、商店 |

| 金(传说) | 10 | 隐藏Boss、通关奖励 |
| 红(禁忌) | 5 | 特殊条件解锁 |

新增机制型功法：
- **蓄力型**：打出后下X张牌获得加成
- **条件型**：满足特定条件触发额外效果（如HP<30%时伤害翻倍）
- **变形型**：可在两种形态间切换
- **消耗型**：打出后从本局游戏中移除，换取强力效果
- **共鸣型**：与其他特定功法同时打出时触发组合效果

#### 2.2 五行相生相克

核心互动机制：

```
相生（打出A后B+效果）：
  火生土 → 打出火牌后下一张土牌格挡+2
  土生金 → 打出土牌后下一张金牌伤害+1
  金生水 → 打出金牌后下一张水牌抽牌+1
  水生木 → 打出水牌后下一张木牌治疗+2
  木生火 → 打出木牌后下一张火牌灼烧+1

相克（对克制状态敌人+伤害）：
  火克木 → 对中毒敌人火牌+3伤害
  木克土 → 对有格挡敌人木牌+2伤害
  土克水 → 对有灼烧敌人土牌+2格挡
  水克火 → 对高攻击意图敌人水牌费用-1
  金克木 → 金牌无视中毒类减伤
```

#### 4.4 剧情系统：「凡人修仙录」

### 剧情框架：千年轮回

**核心设定：** 玩家是「林凡」，一个灵根低劣的凡人少年，偶然获得一枚神秘玉佩，开启艰难修仙路。每次历练（进入秘境）都是成长的一步。通关越多，修为越深，身世之谜也一步步揭开。

**三层叙事结构：**

```
第一层：主角线（艰难成长+身世） → 跨局推进，6秘境通关讲完
第二层：世界线（修仙界暗流）    → 通过仙人日志/事件碎片收集
第三层：秘境线（每个秘境的独立故事） → 单层内讲完，6层递进
```

### 主角线：五章剧情

**序章：凡人起步**
- 开场动画：林凡是边陲小镇的孤儿，在破庙中发现一枚温热玉佩
- 玉佩中封印着一位上古修士残魂（师父），开始教他修炼
- 第一次进入幽冥谷，师父说：「活着回来，就算你入门了」
- 基调：**艰难**，只有微弱灵根，一切靠自己拼

**第一章：炼气·机缘**（通关幽冥谷后触发）
- 在幽冥谷深处发现一具上古修士遗骸，获得一枚「筑基丹方」
- 师父惊讶：「这是…失传三千年的丹方。你的运气好得离谱。」
- 偶遇受伤少女「苏瑶」，她是散修，被妖兽追杀。林凡救下她
- 苏瑶成为**道侣**（系统解锁），提供被动加成
- 基调：**转运**，从底层开始往上爬

**第二章：筑基·入世**（通关赤炎洞后触发）
- 苏瑶的身份揭露：她是天机阁阁主之女，门阀势力牵扯进来
- 天机阁追杀苏瑶的仇敌盯上了林凡：「区区凡人，也配修仙？」
- 师父：「你的玉佩…不只是机缘那么简单。它认了你。」
- 基调：**危机**，真正的修仙界残酷一面

**第三章：金丹·身世**（通关碧落林后触发）
- 玉佩碎裂，露出内层铭文——这是上古仙帝「林天」的本命玉牌
- 真相：林凡是仙帝转世。三万年前仙帝对抗天魔，肉身毁灭，仅存一缕残魂封入玉佩转世
- 师父就是仙帝当年的护道者，守护了三万年
- 基调：**震撼**，原来一路的艰难都是转世后的宿命

**第四章：元婴·劫难**（通关寒霜渊后触发）
- 天魔残魂苏醒，万妖山妖皇是天魔的信徒
- 苏瑶被天魔残魂侵蚀，需要前往万妖山找到解药
- 师父耗尽残魂之力为苏瑶暂时压制侵蚀：「我的使命完成了…照顾好她。」
- 师父消散，洞府中留下他的遗物（专属法器）
- 基调：**悲壮**，失去最重要的人

**终章：化神·了结**（通关太虚境后触发）
- 太虚境是天魔残魂的藏身之地
- 最终Boss战：天魔残魂（借用白逸尘肉身）
- 苏瑶在关键时刻以自身为引，助林凡觉醒仙帝记忆
- **两种结局：**
  - 击败天魔 → 恢复仙帝记忆，但选择做回林凡（凡人心比仙帝心更珍贵）→ 解锁「渡劫」难度
  - 隐藏结局 → 若收集全部仙人日志+6秘境全通，可同时救回苏瑶和师父 → 解锁隐藏秘境「天道秘藏」

### 道侣系统：苏瑶

**成长线：** 随剧情推进解锁不同加成

| 阶段 | 触发 | 加成 | 故事 |
|------|------|------|------|

| 携手 | 通关赤炎洞 | 每层开始多抽1张牌 | 「一起走下去吧。」 |

| 患难 | 通关寒霜渊 | 被侵蚀后加成变为随机(强但不稳定) | 「我…还能认出你吗？」 |
| 归来 | 通关太虚境+真结局 | 全加成+复活1次 | 「我回来了。」 |

**道侣战斗技能：** 每场战斗可主动触发1次
- 初遇：回复5HP
- 携手：抽1张牌
- 并肩：格挡翻倍
- 患难：随机(回复10HP/抽2张/格挡翻倍/攻击+5)
- 归来：全部效果

### NPC 系统

**师父（残魂）**
- 洞府常驻，每次通关触发对话
- 台词随境界变化：引导→暗示→揭示→告别
- 终章后消失，留下遗物「护道玉」(法器)

**游方散商（莫老三）**
- 地图商店节点，市侩但有情义
- 5次交易后解锁身世：他曾是仙帝的侍从，转世后失去了记忆，本能地继续行商
- 卖「前世遗物」(随机强力道具)，价格随通关次数递增

**灵狐（小九）**
- 碧落林专属，引导NPC+支线
- 完成支线 → 永久获得「灵狐指引」被动

### 仙人日志（30条）

前世记忆碎片，每条是一段短文：

> **「忘川散人·手记 其三」**
> 天启三年，与苏瑶游历东海。遇一老龟，已活万年，化形失败七次。我问它为何执着，它说：「百年修得一朝形，千年只求一回眸。」苏瑶笑它痴愚。我总觉得，她笑的不是龟。

**获取方式：**
- 首次通关每层秘境：30%
- 精英击杀：15%
- 特殊事件：100%
- 隐藏节点：50%

**收集进度：**
```
仙人日志：0/30
 ├─ 卷一·少年行（1-10）  → 解锁身世背景
 ├─ 卷二·入道途（11-20）  → 解锁角色关系
 └─ 卷三·忘川劫（21-30）  → 解锁真结局线索
```

### 秘境线：每层独立叙事

以幽冥谷为例（6层递进故事）：

| 层数 | 叙事节点 |
|------|----------|
| 1层 | 进入时：「这里曾有人渡劫失败…空气中弥漫着残余灵力。」 |
| 2层 | 墙壁上发现模糊符文，师父辨认出是封印术 |
| 3层 | 无面行者低语：「你不属于这里…但你也无处可去。」 |
| 4层 | 发现渡劫失败者的遗书：「我本可成功…有人从背后…」 |
| 5层 | Boss有短暂清晰面孔——是一张与林凡相似的脸 |
| 6层 | 通关后获得仙人日志+师父：「这个秘境比我想的更危险。你准备好了吗？」 |

### 对话设计原则

1. **每次不超过3句** — 不耽误肉鸽节奏
2. **文字版演出** — 不需配音，背景变化+文字足够
3. **可跳过** — 所有剧情可跳过，不影响老玩家
4. **碎片化** — 30条日志+5段主线+6个秘境线，不集中轰炸
5. **二刷有新东西** — 第二次打同一层，NPC说「你又来了」，解锁新对话
6. **艰难感贯穿** — 主角从底层爬起，每次突破都有代价，不是爽文是成长

  - 首通幽冥谷 → 解锁赤炎洞 + 新功法5张
  - 首通赤炎洞 → 解锁碧落林 + 新神通3个

## 十五、设计原则

1. **外围养成驱动重玩** — 每次历练获得材料，回来强化角色，下次走得更远
2. **多系统联动** — 境界→神通、材料→丹药/法器、功法→牌组、灵根→全链路
3. **选择即策略** — 丹药装备上限、法器升级消耗、功法互斥、神通位有限
4. **渐进难度** — 敌人随层数缩放、Boss狂暴线、后期需综合养成才能通关
5. **五行深度** — 相生相克让每张功法的选择都有策略意义
6. **差异化** — 修仙+卡牌肉鸽+洞府养成三合一，Steam无同类产品
