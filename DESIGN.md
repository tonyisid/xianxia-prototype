## 一、项目概述

**类型**：修仙题材卡牌肉鸽（Roguelike Deckbuilder）
**平台**：网页单文件 HTML（未来可移植微信小程序）
**技术栈**：Vanilla JS + CSS（零依赖）
**核心循环**：外围养成（洞府）→ 卡牌肉鸽副本（幽冥谷）→ 材料回归 → 养成强化

### 游戏流程

```
[首次进入] → 灵根觉醒（随机五行）
     ↓
[洞府/主界面] ←—————————————┐
  ├── ⚔️ 历练 → 准备丹药 → 幽冥谷副本 → 通关/死亡 → 返回洞府
  ├── 🧘 修炼 → 积攒修为 → 突破境界（三选一神通）
  ├── ⚗️ 炼丹 → 消耗材料 → 炼制丹药（装备生效）
  ├── 🔨 炼器 → 消耗材料 → 炼制法器（一阶→二阶升级）
  ├── 📖 功法阁 → 查看50+功法卡池/已收集卡牌
  ├── ⚡ 神通 → 装备被动战斗技能（最多3个）
  ├── 🌅 悟道 → 消耗200修为领悟神通（三选一）
  └── 📦 背包 → 查看装备/材料/重置存档
```

## 二、角色系统

### 2.1 灵根（五行）

首次进入游戏随机分配，决定初始优势和专属卡牌。

| ⚔️ 金 | #bdc3c7 | 10%暴击（双倍伤害），多1张金刃符 | — |

### 2.2 境界体系

共15个境界等级，每层需积攒修为突破。

```
炼气1层(0xp) → 2层(100) → 3层(200) → 4层(350) → 5层(500)
→ 6层(700) → 7层(1000) → 8层(1400) → 9层(1900)
→ 筑基初期(2500) → 中期(3200) → 后期(4000)
→ 金丹初期(5000) → 中期(6200) → 后期(7500)
```

**突破消耗：**
- 破层（每层）：💊 突破丹 ×1
- 破境（每3层）：✨ 筑基丹 ×1

**突破奖励：**
- 每次突破：maxHp +5
- 每3层突破：maxSp +1
- 突破时三选一获得1个神通

### 2.3 持久化存档

使用 localStorage（key: `xianxia_save`），保存内容：
- 灵根、境界、修为
- 材料（灵草/矿石/妖丹）
- 突破丹/筑基丹库存
- 已学习功法、已装备功法
- 已领悟神通、已装备神通
- 已拥有丹药/法器及装备状态
- 历练卡牌收藏
- 悟道冷却时间

## 六、功法卡牌系统

**功法即卡牌。** 所有战斗中打出的牌都是功法，每张功法带五行属性。

### 6.1 卡牌数据结构

```typescript
{
  id: number;          // 唯一ID
  name: string;        // 名称
  type: 'atk'|'def'|'skill';  // 攻击/防御/功法
  cost: number;        // 灵力消耗
  dmg: number;         // 伤害
  blk: number;         // 格挡
  burn: number;        // 灼烧
  poison: number;      // 中毒
  draw: number;        // 抽牌
  heal: number;        // 治疗
  el: Element;         // 元素属性(fire/wood/earth/water/metal/none)
  desc: string;        // 描述
  upgraded: boolean;   // 是否升级
}
```

### 6.2 元素属性

| 元素 | 标签 | 定位 | 关联机制 |
|------|------|------|----------|
| 🔥 火(fire) | 符箓·火 | 进攻型 | 灼烧、高伤害 |
| 🌿 木(wood) | 符箓·木 | 持续型 | 中毒、回复 |
| ⛰️ 土(earth) | 符箓·土 | 防御型 | 格挡、攻防兼备 |
| 💧 水(water) | 符箓·水 | 控制型 | 抽牌、攻守平衡 |
| ⚔️ 金(metal) | 符箓·金 | 暴击型 | 高直伤、穿透 |
| 无(none) | 功法·无 | 中立 | 通用辅助 |

## 七、战斗系统

### 7.1 战斗流程

```
startTurn()
  → 灵力恢复、格挡清零、神通回合初始化
  → 抽牌（5张 + 水灵根/聚灵佩加成）
  → 显示敌人意图

[玩家出牌阶段]
  → _playCard() → 费用扣除 → 神通触发检查 → 效果结算
  → 可用丹药（每场1次）
  → 点击「结束回合」

endPlayerTurn()
  → 弃手牌
  → 灼烧tick（伤害+层数-1，百毒侵不掉）
  → 敌人行动（意图结算）
  → 以柔克刚检查
  → 不灭金身检查
  → 中毒tick
  → 涅槃检查（死亡复活）
  → 蓄势追踪
```

### 7.2 伤害计算

```
总伤害 = 基础伤害
  + 火鸦加成（火鸦变）
  + nextAtk（淬火术/凝元丹）
  + 丹意层数
  + 灵根加成（火+1）
  + 功法加成
  + 法器加成（玄铁剑）
  + 神通加成（舍身/烈焰之心/弱点洞悉）

  × 焚天（第3张火翻倍）
  × 金锐（15%翻倍）

  × 金灵根暴击（10%）

```

### 7.3 状态效果

| 状态 | 触发时机 | 效果 |

## 十二、设计原则

1. **外围养成驱动重玩** — 每次历练获得材料，回来强化角色，下次走得更远
2. **多系统联动** — 境界→神通、材料→丹药/法器、功法→牌组、灵根→全链路
3. **选择即策略** — 丹药装备上限、法器升级消耗、功法互斥、神通位有限
4. **渐进难度** — 敌人随层数缩放、Boss狂暴线、后期需综合养成才能通关

## 十三、Steam 版开发路线图

从浏览器原型扩展为 Steam 单机游戏，分四个阶段。

### Phase 1：基础建设（2-3月）

**目标：** 可运行的独立游戏框架

#### 1.1 引擎迁移
- 选择引擎：推荐 Godot 4（免费、轻量、2D友好、导出 Steam 方便）
- 备选：Unity（生态成熟但重）、自研引擎（仅适合有经验者）
- 迁移所有核心系统：卡牌、战斗、敌人AI、状态效果
- 重构数据层：JSON/Resource 文件管理功法/敌人/丹方配置

#### 1.2 美术规范
- 确定美术风格：水墨国风（推荐）或 像素修仙 或 日式卡牌
- 需要的美术资源：
  - 卡牌插画：50张功法 × 2（普通/升级）= 100张
  - 敌人立绘：30+种
  - UI 框架：洞府/战斗/弹窗
  - 背景：洞府/秘境×4/休整点
  - 特效：火焰/中毒/格挡/暴击/突破
- 美术外包预算参考：3-8万（水墨风）

#### 1.3 音效音乐
- BGM（至少4首）：洞府(宁静)、普通战斗(紧张)、精英/Boss(史诗)、休整(舒缓)
- 音效（30+）：出牌/受伤/死亡/突破/炼丹/炼器/UI点击/抽牌/弃牌
- 预算参考：1-3万（外包）

#### 1.4 Steam 集成
- Steam SDK 接入
- 成就系统（20-30个）
- 云存档
- 排行榜（每日挑战）

### Phase 2：内容扩充（2-3月）

**目标：** 足够的内容支撑 10+ 小时游戏时长

#### 2.1 功法扩充（150+张）

当前50张 → 目标150+张，每个元素25-30张：

| 稀有度 | 数量 | 获取方式 |
|--------|------|----------|
| 白(普通) | 60 | 普通战斗、商店 |
| 绿(进阶) | 50 | 精英战斗、事件、商店 |
| 蓝(稀有) | 25 | Boss、稀有事件 |
| 金(传说) | 10 | 隐藏Boss、通关奖励 |
| 红(禁忌) | 5 | 特殊条件解锁 |

新增机制型功法：
- **蓄力型**：打出后下X张牌获得加成
- **条件型**：满足特定条件触发额外效果（如HP<30%时伤害翻倍）
- **变形型**：可在两种形态间切换
- **消耗型**：打出后从本局游戏中移除，换取强力效果
- **共鸣型**：与其他特定功法同时打出时触发组合效果

#### 2.2 五行相生相克

核心互动机制：

```
相生（打出A后B+效果）：
  火生土 → 打出火牌后下一张土牌格挡+2
  土生金 → 打出土牌后下一张金牌伤害+1
  金生水 → 打出金牌后下一张水牌抽牌+1
  水生木 → 打出水牌后下一张木牌治疗+2
  木生火 → 打出木牌后下一张火牌灼烧+1

相克（对克制状态敌人+伤害）：
  火克木 → 对中毒敌人火牌+3伤害
  木克土 → 对有格挡敌人木牌+2伤害
  土克水 → 对有灼烧敌人土牌+2格挡
  水克火 → 对高攻击意图敌人水牌费用-1
  金克木 → 金牌无视中毒类减伤
```

#### 2.3 秘境扩充（4个）

| 秘境 | 主题 | 节点数 | 特殊机制 | Boss |
|------|------|--------|----------|------|
| 幽冥谷 | 阴暗、蝙蝠 | 12 | 多目标战斗 | 巨蝠妖 |
| 赤炎洞 | 火山、熔岩 | 14 | 全局灼烧(每回合受伤) | 炎魔 |
| 碧落林 | 森林、毒瘴 | 14 | 中毒陷阱节点 | 万毒蛛母 |
| 太虚境 | 虚空、幻象 | 16 | 随机debuff、镜像战斗 | 虚空之主 |

每个秘境解锁条件：
- 幽冥谷：默认开放
- 赤炎洞：通关幽冥谷 + 筑基境界
- 碧落林：通关赤炎洞 + 金丹境界
- 太虚境：通关碧落林 + 收集3件传说法器

#### 2.4 敌人扩充（30+种）

每个秘境 6-8 种普通敌人 + 2-3 种精英 + 1 Boss：
- 幽冥谷：现有5种 + 新增暗影蝠、鬼面蛛、冥河蛇
- 赤炎洞：熔岩史莱姆、火焰蜥蜴、岩浆元素、灰烬亡魂等
- 碧落林：毒蘑菇精、藤蔓魔、花妖、蛊虫群等
- 太虚境：虚空行者、镜像僧、时空裂隙、混沌兽等

#### 2.5 神通扩充（30+种）

当前17种 → 目标30+：
- 新增秘境专属神通（通关特定秘境解锁）
- 新增五行共鸣神通（需要装备同系功法触发）
- 新增组合神通（两个低级神通融合为高级）
